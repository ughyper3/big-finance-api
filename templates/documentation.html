    example : {Authorization: Token qwlifowajfaiejfie12904e913wqfdjiwqfoij1298e}
    To get a token, please contact me.

  • Endpoints :

    • Finnhub data api


      profile : https://bigfinance-api.herokuapp.com/finnhub/api/v1/profile/COMPANY SYMBOL/
      quote : https://bigfinance-api.herokuapp.com/finnhub/api/v1/quote/COMPANY SYMBOL/
      recommendation : https://bigfinance-api.herokuapp.com/finnhub/api/v1/recommendation/COMPANY SYMBOL/

      news : https://bigfinance-api.herokuapp.com/finnhub/api/v1/profile/NEWS CATEGORY/ news category can be general, forex, crypto or merger
